# Debugging Alternatives to GDB

## GUI Debuggers
- **kdbg** - KDE debugger frontend for GDB
- **qtcreator** - Qt Creator IDE with integrated debugger
- **ddd** - Data Display Debugger, graphical frontend for GDB
- **cgdb** - Curses-based terminal debugger frontend for GDB
- **nemiver** - GNOME graphical debugger frontend for GDB
- **kdevelop** - KDE IDE with integrated debugger

## Command-Line Debuggers
- **lldb** - LLVM debugger (alternative to GDB, used by Xcode)
- **rr** - Record and replay debugging tool
- **radare2** - Reverse engineering framework with debugging capabilities

## Memory & Performance Tools
- **valgrind** - Memory debugging and profiling tool
- **AddressSanitizer (ASan)** - Fast memory error detector
- **perf** - Linux performance analysis tool
- **strace/ltrace** - System call and library call tracing

## Commercial/Proprietary
- **IDA Pro** - Interactive disassembler and debugger
- **WinDbg** - Windows debugger
- **Visual Studio Debugger** - Microsoft Visual Studio integrated debugger
- **TotalView** - Commercial parallel debugger

## Other Tools
- **gdb-dashboard** - Python extension for GDB with better visualization
- **pwndbg** - GDB plugin for exploit development
- **gef** - GDB Enhanced Features plugin
