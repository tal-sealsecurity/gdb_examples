run: starts or restarts program execution  
break: sets a breakpoint at specified location  
next: executes next line, stepping over function calls  
list: displays source code around current location  
print: prints the value of a variable or expression  
quit: exits GDB debugger  
step: executes next line, stepping into function calls  
continue: resumes program execution until next breakpoint  
watch: sets a watchpoint to monitor variable changes  
up: moves up one frame in the call stack  
down: moves down one frame in the call stack  
backtrace: displays the call stack (function call chain)  
finish: executes until current function returns  
display: automatically displays expression value at each stop  
undisplay: stops automatic display of an expression  
info breakpoints: shows information about all breakpoints  
info locals: displays local variables in current scope  
delete: removes breakpoints or watchpoints  
layout: changes GDB's display layout (src, asm, split, etc.)  
target record-full: enables reverse debugging (record execution)  
reverse next: steps backward to previous line  
ref: refreshes the display
